# reThink Auto-Capture Extension

Browser extension para capturar automáticamente interacciones de WhatsApp Web y LinkedIn.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El output estará en `dist/`. Para cargar en Chrome:
1. Abrir `chrome://extensions`
2. Habilitar "Developer mode"
3. Click "Load unpacked"
4. Seleccionar carpeta `dist/`
