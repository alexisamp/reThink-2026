# Ambient Sound Files

The Focus Timer uses these audio files for ambient sound:

- `brown-noise.mp3` — Brown noise (deeper, warmer than white noise)
- `rain-noise.mp3` — Rain / nature sounds

## Sources (free/royalty-free)
- https://mynoise.net — Brown noise generator
- https://freesound.org — Search "brown noise loop" or "rain loop"
- https://pixabay.com/sound-effects/ — Search "brown noise" or "rain ambience"

Download loops (~2-5 min, stereo MP3, ~128kbps is fine) and place them here.
The app loops them automatically at 30% volume during focus sessions.
